# Skill: react

Frontend expertise with React and the modern component model.

## Competence by weight

- Low (0.0-0.3): can wire a basic component, misses idioms.
- Mid (0.4-0.6): solid components, hooks, and state.
- High (0.7-0.9): performance, rendering behavior, composition, and ecosystem fluency.

## Pairs well with

- `analytical` and `defensive` for robust UI that handles loading and error states.
- the `state-sync` style of problems, real-time and interactive interfaces.
