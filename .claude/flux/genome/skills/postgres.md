# Skill: postgres

Database knowledge centered on PostgreSQL: schema, queries, and performance.

## Competence by weight

- Low (0.0-0.3): basic tables and queries.
- Mid (0.4-0.6): sound schema, indexes, joins, transactions.
- High (0.7-0.9): query plans, partitioning, concurrency, performance tuning.

## Pairs well with

- `cautious` and `methodical` for migrations and data integrity.
- `performance`-oriented problems where the bottleneck is the data layer.
